# mem-vibe

Minimal React + TypeScript app (Vite). The UI is intentionally empty: a blank page.

## Prerequisites

- [Node.js](https://nodejs.org/) 20+ (LTS recommended)

## Setup

Install dependencies once:

```bash
npm install
```

## Development

Start the dev server with hot reload (default: http://localhost:5173):

```bash
npm run dev
```

## Production build

Type-check, bundle, and emit static files to `dist/`:

```bash
npm run build
```

Serve the production build locally to verify:

```bash
npm run preview
```

## Lint

```bash
npm run lint
```
